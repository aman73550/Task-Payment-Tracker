### 💳 UPDATE: UDHAAR (LENDING & BORROWING) MODULE

AI Agent ko ye specific instructions dein:

#### 1. Database Update (New Table: `udhaar`)
Parchhi (Tasks) se alag ek table banayein taaki accounting saaf rahe:
```sql
create table udhaar (
  id uuid default uuid_generate_v4() primary key,
  user_id uuid references auth.users not null,
  person_name text not null,
  amount decimal not null,
  type text check (type in ('Give', 'Take')), -- Give: Aapne udhaar diya, Take: Aapne liya
  status text check (status in ('Active', 'Settled')) default 'Active',
  due_date timestamp with time zone,
  history jsonb default '[]'::jsonb, -- Settlement records: [{date, amount_paid, note}]
  created_at timestamp with time zone default now()
);
```

#### 2. The "Udhaar Dashboard" (High-End View)
Finance section ke andar ek alag tab hoga "Udhaar" naam se:
* **Total To Receive (Aane Wala):** Jitna aapne udhaar diya hai (Give). Isko **Gold (#D4AF37)** mein dikhayein.
* **Total To Pay (Dena Wala):** Jitna aapne udhaar liya hai (Take). Isko **Soft Grey (#A3A3A3)** mein dikhayein.
* **Net Balance:** Dono ka difference, taaki aapko apni real position pata rahe.

#### 3. Core Features: Add, Edit & Settle
* **Add Udhaar:** Ek simple form jisme *Person Name*, *Amount*, *Type (Lent/Borrowed)* aur *Due Date* ho.
* **Quick Settle Button:** Har record ke bagal mein ek "Settle" button ho. Agar koi partial payment (thoda paisa) wapas kare, toh wo seedhe `history` mein timestamp ke saath save ho jaye.
* **Status Toggle:** Ek baar poora paisa settle ho jaye, toh entry ko "Settled" mark kar dein (ye main list se hat kar 'History' mein chali jayegi).

#### 4. Design & Human-Touch
* **Visual Cues:** Jo paisa aapko milna hai uske aage up-arrow (↗) aur jo dena hai uske aage down-arrow (↘).
* **Conversational Reminders:** App aapko yaad dilaye: *"Aman, you need to collect ₹5,000 from [Name] today."*
* **Flat Icons:** `ArrowUpRight`, `ArrowDownLeft`, `HandCoins` (1.5px stroke, Pearl White).
